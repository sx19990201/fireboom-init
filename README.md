# fireboom startup shell

## Install

```shell
git clone https://github.com/sx19990201/fireboom-init.git
cd fireboom-init
./fireboom.sh init
```

## Run

```shell
./fireboom.sh
# or run "./fireboom.sh init" to re-init
```

## Update

```shell
./fireboom.sh update
```

## Show versions

```shell
./fireboom.sh version
```
